MyPod - dsPIC Based MP3 Player

Project Description

This is a simple single-chip MP3 player using a dsPIC33E512GP502. It should also work with a dsPIC33F with some modifications to the code (the DMA on the dsPIC33F can only access dual-ported RAM (up to 4KB) which is not enough for 2x2 channel buffers (for ping-pong), so we'll have to modify the code to deal with that. Besides the dsPIC with used an ILI9341 LCD (about $6 on eBay) and an op-amp and microphone for recording, plus a few discrete components. The software source code is available on GitHub (see link below) and schematics will be available soon).
