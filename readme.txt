This project is a work in progress. Only the source code
is available at this time. You will need to figure out how
to build the board to use it. It shouldn't be too hard since
only a few connections are required and they're all defined
in the source code. A schematic will be available in the
future.


Software Requirements:
======================

1. MP3Lib Decoder Library - Free download (personal use) 
   from www.fernansoft.com/mp3lib.aspx

2. Fat32Lib - Free download (GPL) from 
   www.fernansoft.com/fat32lib.aspx

3. XC16 Compiler - Free download from microchip.com


Hardware/Component Requirements:
======================

1. dsPIC33F or dsPIC33E with 32KB RAM
2. ILI9341 SPI LCD
3. Breadboard
4. 1 transistor/few resistors and caps
5. 4 push buttons
